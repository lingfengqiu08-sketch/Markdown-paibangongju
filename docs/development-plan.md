# 开发计划

## 项目概要

**项目名称**：Markdown 排版工具  
**开发周期**：3-4 周  
**团队规模**：1-2 人  
**技术栈**：React + Vite + Marked.js + Tailwind CSS

---

## 开发原则

1. **MVP优先**：先实现核心功能，确保基础可用
2. **快速迭代**：每日部署，快速收集反馈
3. **渐进增强**：从简单到复杂，逐步完善
4. **用户驱动**：根据用户反馈调整优先级
5. **保持简洁**：避免过度设计，保持工具简单易用
